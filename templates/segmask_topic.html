3D Bounding Box Prediction

Aim: Adopt a multi-sensor approach to detect 3D objects from RGB + LiDAR data by leveraging a robust 2D object detector like YOLOv7 along with a novel segmentation mask on top of the Frustum PointPillar approach to reduce the search space in 3D detection.

Intuition: 2D image recognition tasks like object detection and instance segmentation, have made significant strides lately but beyond obtaining 2D bounding boxes or pixel masks, 3D comprehension is avidly sought in many major applications including autonomous driving and augmented reality.

Approach: In light of this, there is an increasing demand for 3D object detection models that classify accurately and estimate the oriented 3D bounding boxes of actual objects using multi-sensor data. This work develops on top of Frustum PointPillar by leveraging robust 2D object detectors as priors to build a 3D viewing frustum that predicts a 3D bounding box of the object from the points in the frustum.

  • Modified the existing Frustum-PointPillars architecture which uses images and point-cloud data (sensor fusion camera and LiDAR) by replacing the naive Gaussian mask with a pre-trained PSPNet segmentation mask and achieved a mAP score of 78.01 (+3%)for cars class in the KITTI-Hard dataset.
  • Established a real-time detection by adding a YOLOv7 2D detection layer instead of using KITTI ground truths.
  • The future scope of this work is to improve 2D detection in low-light conditions using data from RGB and LiDAR and to avoid outliers, it is essential to remove reflections in surfaces such as glasses and windows.
