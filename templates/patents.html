Q-CerGen (Quick Certificate Generator)

I innovatively crafted a solution for the swift generation of a significant volume of E-certificates, utilizing an Excel file and a certificate template. These certificates boast E-verification with a QR code implemented through the SHA algorithm. Leveraging OpenCV, I coded the application, and with Tkinter in Python, I designed a user-friendly interface. Additionally, I expanded the impact by developing the Q-CerGen website using Flask, and went a step further to create 3D E-trophies for participants using Brython and webGL.

ADAM (Automatic Disassemble and Assemble Machine)

I engineered a prototype robotic pick-and-place arm with the ability to disassemble a pattern composed of basic blocks and reassemble it elsewhere. The project involved coding for object recognition and pattern mapping, employing the SIFT algorithm to dynamically map diverse blocks constituting a specific pattern. Additionally, I crafted a 3D graphical user interface (GUI) using OpenGL and Python for customizing patterns. To further enhance functionality, I integrated a machine learning model to assess the structural stability of the assembled patterns.
